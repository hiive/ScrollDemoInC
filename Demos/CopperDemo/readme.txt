Copper Demo
===========

This demo enables the loRes screen and it is cleared to colour index 0.
The Copper is programmed each frame to change the colour of colour index 0 
for n number of scanlines.
The user can increase or decrease the number of scanlines to change per screen refresh. 

The blue border at the bottom shows how much time is being taken 
to program the copper to make the colour changes.

Keys
    p = increase number of copper lines
    l = decrease number of copper lines
